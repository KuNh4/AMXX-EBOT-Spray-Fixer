# AMXX EBOT Spray Fixer
This AMX Mod X plugin was made for E-BOT 1.07 LEGACY R12.
It makes bots spray when holding automatic weapons instead of tap/burst shooting.

[E-BOT Legacy R12](https://github.com/EfeDursun125/CS-EBOT-LEGACY)

# CVars
- ebot_spray_duration "1.5" // Duration of the Spray

# Note
There is probably room for improvement on this plugin but I am happy with the functioning of it.
E-BOT Legacy R12 is known to be the most stable version to use with Zombie Mods in CS 1.6.
